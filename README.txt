Для работы парсера необходимо создать базу данных и внести изменения в config.php
По умолчанию:
DB_HOSTNAME = '127.0.0.1';
DB_DATABASE = 'rozetka';
DB_USERNAME = 'rozetka';
DB_PASSWORD = rozetka';

После создания базы необходимо запустить install.php для создания таблицы в базе данных.
Для парсинга сайта rozetka.com.ua нужно запустить index.php.
Скрипты нужно запускать используя PHP CLI.
